Jalankan Scraping Data Kosmetik

{# Judul warna pink #}

Tekan tombol di bawah untuk memulai proses scraping data produk dari BeautyHaul (beberapa merek). Proses ini mungkin memakan waktu beberapa menit.

Panduan Penggunaan:

{# Judul panduan warna pink #}
  1. Pastikan Anda sudah login ke aplikasi.
  2. Klik tombol "Jalankan Scraping Sekarang" di atas.
  3. Proses scraping akan berjalan di latar belakang (Anda mungkin tidak melihat browser muncul jika mode 'headless' aktif).
  4. Setelah proses selesai, akan muncul notifikasi keberhasilan atau kegagalan.
  5. Produk yang berhasil di-scrape akan ditambahkan ke database dan dapat dilihat di halaman Home atau melalui fitur pencarian.
    6. {# Link Home warna pink #}
  6. Riwayat aktivitas scraping Anda juga akan tercatat di halaman Profil Anda.
